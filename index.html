CSS Feature Used: `filter` and `transition`

The profile picture above demonstrates the use of the filter and transition CSS properties. Initially, the image is in grayscale (filter: grayscale(100%)), and when hovered over, it smoothly transitions to full color (filter: grayscale(0%)) over 0.3 seconds using transition: filter 0.3s ease. This technique enhances the interactivity of the webpage while maintaining a clean and engaging design.

HTML Feature Used: `details` and `summary`

The "More About Me" section above uses the <details> and <summary> elements. This is a built-in HTML feature that allows content to be collapsible. When the user clicks the "More About Me" summary, additional information is revealed, and clicking again hides it. This feature enhances user experience by keeping the page clean while still providing access to additional content.
